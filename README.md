# SWAPP Frontend (Vercel Ready)

Pronto para deploy direto na Vercel.

## Rodar local
```bash
npm i
npm run dev
```

## Deploy na Vercel
1. Crie projeto e conecte ao repositório, ou faça _Import_ do zip.
2. Framework: **Next.js**
3. Build Command: `next build`
4. Output: **Next.js default**
5. Node: 18.x, 20.x ou 22.x
6. Domínio: aponte `app.seu-dominio` como **CNAME** para o valor sugerido no painel da Vercel.
7. O middleware redireciona `/` para **/login**.
