SWAPP-frontend – Patch para corrigir 404 na Vercel

Arquivos incluídos (relativos ao repo):
- middleware.ts  (raiz do projeto, ao lado do package.json)
- next.config.mjs (raiz)
- src/app/layout.tsx
- src/app/page.tsx
- src/app/login/page.tsx
- src/app/globals.css

Como aplicar:
1) Apague qualquer pasta `src/pages` se ainda existir.
2) Copie os arquivos deste patch para os mesmos caminhos do seu repositório.
3) Commit & push:
   git add .
   git commit -m "fix: 404 na Vercel (middleware raiz + redirect / -> /login)"
   git push origin main
4) Abra o domínio na Vercel. Deve carregar a tela de Login placeholder.

Se preferir redirecionar a home para /monsters, altere '/login' para '/monsters' nos arquivos:
- middleware.ts
- src/app/page.tsx
